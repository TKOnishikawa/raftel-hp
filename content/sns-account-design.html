投稿テンプレート — 3つの型

A. 実践共有

週4回 / メインの投稿タイプ

今日、製造業のクライアントの在庫管理をAIで自動化してみた。 結果 → 月末棚卸しの作業時間が8時間→1.5時間に。 ポイントは「完璧を目指さない」こと。 まず80%の精度で回して、人間が例外だけ処理する。 これだけで現場の負担が激減する。
B. 専門洞察

週2回 / 知見と経験を凝縮

中小企業のAI活用でよく見る間違い： 「まずChatGPTの使い方を社員に研修しよう」 でも実は、AIツールの研修より先にやることがある。 それは「どの業務にAIを入れるか」の選定。 アクセンチュア時代に100社以上見てきた経験から言うと、 ここをすっ飛ばすと「研修したけど誰も使わない」になる。
C. ツッコミ/本音

週1回 / 共感と差別化

正直に言う。 「AI導入で業務効率50%アップ！」って記事をよく見るけど、 あれ、条件が揃った大企業の話がほとんど。 中小企業は ・データが整理されてない ・専任担当がいない ・予算も限られてる だからこそ「小さく始めて、効果が出た業務から広げる」が鉄則。 派手な数字より、地道な一歩が最強。

Cross-Channel

導線設計 — X → HP → リード獲得

各チャネルは単体で完結させず、ゴール（リード獲得）に向かう一本の導線として設計する。 ポイントは「全ての投稿にリンクを貼らない」こと。Xアルゴリズムはリンクなし投稿を優遇する。

1

X投稿で認知獲得

週7回投稿 / うちリンク付きは1-2回のみ

実践共有・専門洞察・本音ツッコミの3パターンで発信。リンクなし投稿でインプレッションを稼ぎ、週1〜2回だけ「詳しくはHPで」とHP記事に誘導する。

2

HP記事で信頼構築

月2本ペースで記事公開

X投稿では書ききれない深い知見を記事化。「この人は本物だ」と確信させる。記事末尾には必ずリードマグネットのCTAを設置する。

3

リードマグネットでリスト獲得

「中小企業AI活用チェックリスト」無料DL

メールアドレスと引き換えに有益な資料を提供。「チェックリスト」「テンプレート」「事例集」など、すぐ使える実用資料が効果的。

4

自動メール3通でナーチャリング

DL後3日間で段階的に関係構築

1通目: チェックリストの使い方。2通目: よくある失敗パターン。3通目: 無料相談のご案内。セールスではなく「もっと知りたい」を引き出す設計。

5

無料相談 → グロースアドバイザリー提案

ヒアリング30分 / 課題の明確化

相談の場で「御社の場合はこうすべき」と具体的な方向性を提示。その場で売り込まず、「一緒にやりたい」と思ってもらうことがゴール。

  • X投稿には毎回HP記事へのリンクは貼らない（週1〜2回のみ）。リンクなし投稿のほうがXアルゴリズムで優遇される。
  • HP記事には必ずリードマグネットCTAを設置する。記事を読み終えたタイミングが最もモチベーションが高い。
  • リードマグネットは「すぐ使える実用資料」に限定する。ホワイトペーパーは中小企業には響かない。
  • 自動メールは3通まで。それ以上は「しつこい」と感じられてブランド毀損になる。
  • 無料相談のCTAは「まずは30分、御社の状況をお聞かせください」の温度感。ハードルを下げる。

導線の鉄則:
リンクなし投稿で認知を取り、
リンク付き投稿で導線を作り、
リードマグネットで関係を始める。

YouTube

YouTube — 開設時の設定メモ

YouTubeは2ヶ月目以降に開設。Xで発信の型が安定し、HP記事のストックが3本以上溜まってからが最適なタイミング。動画はテキストより圧倒的に信頼構築が速い。

Channel オモシク｜中小企業のAI活用
Description AI活用の裏側、全部見せます。中小企業のDX・データ活用・AI導入を実践で解説。
Thumbnail ダークネイビー背景 × コーラルのタイトル文字（ブランドカラー統一）

コンテンツフォーマット

ロング動画（10-15分）

自声での解説 + 画面収録。HP記事のテーマを動画に展開。「顔が見える」ことで信頼度が大きく変わる。

月2本ペース

ショート（60秒）

AI音声 + スライドで量産。X投稿の内容を60秒にまとめ直す。制作コストを抑えつつリーチを広げる。

週2本ペース

開設OKの条件（すべて満たしたら）:

  • Xの投稿ルーティンが2週間以上継続できている
  • HP記事が3本以上公開済み
  • 動画制作に週3時間を確保できる
  • 動画のネタが10本以上リストアップできている

早すぎる開設のリスク: Xの投稿が止まる、記事更新が止まる、チャネルが全て中途半端になる。順番を守ることが最重要。

Future Channels

将来の拡張 — 実績が溜まってから

最初から全チャネルを同時に始めると、すべてが中途半端になる。1つのチャネルが安定してから次へ。これはSNSマーケティングの鉄則であり、リソースが限られる中小企業では特に重要。

Facebook

起業家グループでの信頼構築 + 既存人脈活用。中小企業経営者はFacebookの利用率が依然高い。コミュニティ参加→有益コメント→自分の投稿、の順で存在感を作る。

3ヶ月目〜

LinkedIn

アクセンチュア経歴の権威性を最大活用。B2B経営者層へのリーチ。英語圏も視野に入れた発信が可能。X投稿を丁寧にリライトして転載するだけでも十分に差別化できる。

3ヶ月目〜

Instagram

制作実績のビジュアル証明。カルーセル投稿中心。Before/After形式でAI導入の成果を視覚化する。デザイン性の高い投稿が求められるため、テンプレートを事前に用意しておく。

実績3件以上蓄積後

Xキャラアカウント

カジュアル層への間口拡大、テスト投稿用。実名アカウントでは出しにくい実験的な投稿やカジュアルな話題を扱う。匿名だからこそのフランクな発信でファン層を広げる。

実名が安定した後

チャネル拡張の原則:

1つのチャネルが安定してから次へ。
同時多発は全部中途半端になる。

X → HP → YouTube → Facebook/LinkedIn → Instagram
この順番を守ること。
